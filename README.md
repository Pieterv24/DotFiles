# Dotfiles repo

### Requirements
- [dotgit](https://github.com/kobus-v-schoor/dotgit)

## Useful commands

clone repo  
`git clone --recurse-submodules https://github.com/Pieterv24/dotfiles`

commit changes  
`dotgit -v commit`

filelist is edited  
`dotgit -v update`

link files from repo to machine  
`dotgit -v restore`

Change encryption password  
`dotgit -v passwd`
